Starlane Runner is a small, polished **HTML5/PixiJS infinite dodger** built as a portfolio piece.

- **Tech**: PixiJS 7, HTML5 canvas, responsive scaling
- **Mechanics**: endless asteroid field, difficulty ramp, score based on survival time
- **Input**: mouse / touch (follow pointer) and keyboard (arrows / WASD)

## Running locally

1. Open `index.html` in a modern browser, **or** serve the folder with any static web server.
2. Click / tap the central button to start.
3. Move the ship to dodge asteroids and survive as long as you can.
